# 业务功能：新建货品

## 需求描述
作为货品管理员，我希望能添加一个新货品，以便后续管理库存。

## 验收标准
1. 登录后台后，能进入货品管理页面。
2. 点击“新建货品”，弹出一个表单。
3. 填写货号（不能重复，可以包含 # 和 -）、选择分类、设置销售价，点击保存。
4. 保存后，看到“创建商品成功”的提示，并且新增的货品出现在货品列表里。

## 测试场景
### 场景1：正常创建货品（happy path）
- **前置条件**：已登录后台，具有货品管理权限。
- **步骤**：
  1. 进入货品管理页面。
  2. 点击“新建货品”。
  3. 输入唯一货号（如 `A#123-456`）。
  4. 选择分类“外套”。
  5. 选择套装“非套装”。
  6. 输入销售价A为“5”。
  7. 点击“确定”保存。
- **预期结果**：
  - 出现“创建商品成功”提示。
  - 弹窗关闭。
  - 货品列表中出现该货号。
